## Основные Команды

### Для установки и запуска проекта необходимо выполнить команды
```shell
npm install
npm run dev
```

### Сборка проекта
```shell
npm run build
```

### Запуск preview-сервера
```shell
npm run preview
```

### Форматирование кода
```shell
npm run prettier
```
